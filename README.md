# README

These are my prusaslicer configurations.

## Location

The prusa configurations are placed in different places, depending on operating system
* Linux: `~/.config/PrusaSlicer` or `~/.var/app/com.prusa3d.PrusaSlicer/config/PrusaSlicer` depending on
  how it was installed.
* Mac: `~/Library/Application Support/PrusaSlicer`
* Windows: `%appdata%/PrusaSlicer`
